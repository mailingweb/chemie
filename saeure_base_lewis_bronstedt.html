1. Säure-Base-Theorien im Vergleich

Theorie Definition Säure Definition Base Beispiel
Brønstedt Protonendonator (H+-Geber) Protonenakzeptor (H+-Aufnehmer) HCl + H2O → H3O+ + Cl-
Lewis Elektronenpaarakzeptor Elektronenpaardonator BF3 + NH3 → BF3NH3

2. Brønstedt-Säure-Base-Paare & Beispiele

Beispiel 1:
HCl (Säure) + H2O (Base) ⇌ H3O+ (Säure) + Cl- (Base)
Konjugiertes Säure-Base-Paar: HCl/Cl- und H3O+/H2O

Beispiel 2:
NH3 (Base) + H2O (Säure) ⇌ NH4+ (Säure) + OH- (Base)
Konjugiertes Säure-Base-Paar: NH4+/NH3 und H2O/OH-

3. Lewis-Säuren & -Basen

Lewis-Säure: BF3 (Elektronenlücke)
Lewis-Base: NH3 (freies Elektronenpaar)
Reaktion: BF3 + :NH3 → BF3NH3
Anwendung: Komplexbildung, Katalyse

4. Ampholyte (Ammoniak + Nachweis)

Ampholyt: Teilchen, das sowohl als Säure als auch als Base reagieren kann (z. B. H2O, HSO4-, NH3).

Ammoniak als Base:
NH3 + H2O ⇌ NH4+ + OH-
Nachweis: Rotfärbung von Phenolphthalein in wässriger NH3-Lösung (basische Reaktion)

5. Protolyse & Autoprotolyse

Protolyse: Übertragung eines Protons (H+) zwischen zwei Reaktionspartnern.
Autoprotolyse des Wassers:
H2O + H2O ⇌ H3O+ + OH-
Ionenprodukt des Wassers: Kw = c(H3O+) · c(OH-) = 10-14 mol2/L2 (bei 25°C)

6. Berechnungen (Ks, Kw, c, N, n)

Größe Bedeutung Formel Einheit
Ks Säurekonstante Ks = c(H+) · c(A-)/c(HA) mol/L
Kw Ionenprodukt des Wassers Kw = 10-14 mol2/L2
pH neg. dekad. Logarithmus von c(H+) pH = -log(c(H+)) -
c Stoffmengenkonzentration c = n/V mol/L
n Stoffmenge n = m/M mol

Beispiel: Berechnung des pH-Werts einer 0,1 M HCl-Lösung:
c(H+) = 0,1 mol/L → pH = -log(0,1) = 1
