1. Lösen von Salzen und Lösungsenthalpie

Definition

Die Lösungsenthalpie (ΔHLösung) beschreibt die Energie, die benötigt oder freigesetzt wird, wenn 1 Mol eines Salzes in Wasser gelöst wird. Sie setzt sich aus zwei Schritten zusammen:

  1. Gitterenergie (ΔHGitter): Energie, die benötigt wird, um das Ionengitter des Salzes zu zerstören (immer endotherm).
  2. Hydratationsenthalpie (ΔHHydrat): Energie, die freigesetzt wird, wenn Ionen von Wassermolekülen umhüllt werden (immer exotherm).

Die Lösungsenthalpie ist die Summe beider Prozesse:

ΔHLösung = ΔHGitter + ΔHHydrat

Ist ΔHLösung > 0, ist der Lösungsvorgang endotherm (Temperatur sinkt). Ist ΔHLösung < 0, ist er exotherm (Temperatur steigt).

2. Berechnung der Lösungsenthalpie

Formel

ΔHLösung = ΔHGitter + ΔHHydrat

Beispiel: Lösen von Natriumchlorid (NaCl)

Schritt Enthalpieänderung (kJ/mol)
Gitterenergie (NaCl → Na+ + Cl-) +787
Hydratation von Na+ -406
Hydratation von Cl- -364
Lösungsenthalpie (ΔHLösung) +3,9

→ Der Lösungsvorgang von NaCl ist leicht endotherm.

3. Weitere Beispiele für Lösungsenthalpien

Salz ΔHLösung (kJ/mol) Reaktionstyp
NH4NO3 +25,7 Endotherm
NaOH -44,5 Exotherm
KCl +17,2 Endotherm
CaCl2 -82,8 Exotherm

4. Löslichkeitsprodukt (KL)

Definition

Das Löslichkeitsprodukt beschreibt das Gleichgewicht zwischen ungelösten Salz und seinen Ionen in gesättigter Lösung. Für ein Salz AaBb gilt:

AaBb (s) ⇌ a A+ (aq) + b B- (aq)
KL = [A+]a · [B-]b

KL ist eine Konstante und hängt von der Temperatur ab.

Beispiel: Silberchlorid (AgCl)

AgCl (s) ⇌ Ag+ (aq) + Cl- (aq)
KL = [Ag+] · [Cl-] = 1,8 · 10-10 mol2/L2 (25°C)

→ Je kleiner KL, desto schlechter löslich ist das Salz.

5. Berechnung mit dem Löslichkeitsprodukt

Beispiel: Löslichkeit von AgCl

Gegeben: KL = 1,8 · 10-10 mol2/L2

Gesucht: Löslichkeit s in mol/L

s = [Ag+] = [Cl-]
KL = s · s = s2 → s = √KL = √(1,8 · 10-10) ≈ 1,34 · 10-5 mol/L

→ AgCl ist sehr schwer löslich.

6. Zusammenfassung

  • Lösungsenthalpie: ΔHLösung = ΔHGitter + ΔHHydrat
  • Endotherm: ΔHLösung > 0 (z. B. NH4NO3)
  • Exotherm: ΔHLösung < 0 (z. B. NaOH)
  • Löslichkeitsprodukt: KL = [A+]a · [B-]b
  • Anwendung: Berechnung der Löslichkeit schwerlöslicher Salze
