1. Temperaturmessung beim Lösen von NH₄Cl und NaOH

Beobachtung

  • NH₄Cl (Ammoniumchlorid): Temperatur sinkt → endotherme Reaktion (Energieaufnahme)
  • NaOH (Natriumhydroxid): Temperatur steigt → exotherme Reaktion (Energieabgabe)

Auswertung

Substanz Reaktionstyp Enthalpieänderung (ΔH) Systemart
NH₄Cl Endotherm ΔH > 0 Offenes System
NaOH Exotherm ΔH < 0 Offenes System

Reaktionsgleichungen

NH₄Cl (s) → NH₄⁺ (aq) + Cl⁻ (aq)     ΔH > 0 (endotherm)
NaOH (s) → Na⁺ (aq) + OH⁻ (aq)     ΔH < 0 (exotherm)

2. Herstellung von Ethin (C₂H₂) aus Calciumcarbid (CaC₂)

Beobachtung

  • Starkes Aufschäumen bei Wasserzugabe
  • Gasentwicklung (Ethin, C₂H₂) mit gelber, rußender Flamme
  • Reaktion ist stark exotherm

Auswertung

CaC₂ + 2 H₂O → C₂H₂ + Ca(OH)₂     ΔH < 0 (exotherm)

Systemart: Offenes System (Gasentweichung, Energieaustausch)

Protokollvorlage für beide Experimente

  1. Versuchsüberschrift (z. B. "Temperaturmessung bei Lösungsvorgängen")
  2. Aufgabenstellung (Ziel des Experiments)
  3. Geräte & Chemikalien (auflisten)
  4. Durchführung (schrittweise Beschreibung)
  5. Beobachtung (Temperatur, Gas, Flamme etc.)
  6. Auswertung (Reaktionsgleichungen, Systemart, Energetik)
  7. Sicherheitshinweise (Schutzbrille, Abzug, kleine Mengen)

Zusammenfassung der Lerninhalte

  • Exotherme Reaktion: Energieabgabe (ΔH < 0), Temperatur steigt (z. B. NaOH)
  • Endotherme Reaktion: Energieaufnahme (ΔH > 0), Temperatur sinkt (z. B. NH₄Cl)
  • Ethin-Synthese: CaC₂ + H₂O → C₂H₂ + Ca(OH)₂ (exotherm, offenes System)
  • Protokoll: Strukturiert dokumentieren (Beobachtung ≠ Auswertung)
