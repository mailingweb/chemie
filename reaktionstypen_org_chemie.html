Reaktionstypen in der Organischen Chemie – Abiturwissen

Übersicht mit Beispielen, Reaktionsgleichungen und Bedeutung

Substitution Eliminierung Addition Umlagerung Kondensation Halogenisierung Alkylierung Acetylierung Dehydratisierung Dehydrierung Decarboxylierung Dehalogenierung

1. Substitution

Definition: Austausch eines Atoms oder einer Atomgruppe durch ein anderes Atom oder eine andere Atomgruppe.

Bedeutung: Wichtig für die Synthese von Halogenalkanen, Alkylbenzenen, Acetylverbindungen und vielen weiteren organischen Verbindungen.

Beispiele & Reaktionsgleichungen

Typ Beispiel Reaktionsgleichung Bedeutung
Radikalische Substitution (Halogenierung) Methan + Chlor → Chlormethan CH₄ + Cl₂ → CH₃Cl + HCl Herstellung von Halogenalkanen, z. B. für Lösungsmittel oder Zwischenprodukte in der Synthesechemie.
Elektrophile Substitution (Aromaten) Benzol + Brom → Brombenzol C₆H₆ + Br₂ → C₆H₅Br + HBr Einführung von Substituenten in Aromaten, z. B. für die Herstellung von Arzneimitteln oder Pestiziden.
Alkylierung (Friedel-Crafts) Benzol + Chlormethan → Toluol C₆H₆ + CH₃Cl → C₆H₅CH₃ + HCl Synthese von Alkylbenzenen, z. B. für die Herstellung von Tensiden oder Kunststoffen.
Acetylierung Anilin + Acetylchlorid → Acetanilid C₆H₅NH₂ + CH₃COCl → C₆H₅NHCOCH₃ + HCl Schutzgruppenchemie, Herstellung von Arzneimitteln (z. B. Paracetamol).
Quellen: Abiweb, Lernhelfer, Wikipedia

2. Eliminierung

Definition: Abspaltung von Atomen oder Atomgruppen aus einem Molekül, meist unter Bildung von Mehrfachbindungen.

Bedeutung: Wichtig für die Synthese von Alkenen, Alkinen und die Herstellung von Polymeren.

Beispiele & Reaktionsgleichungen

Typ Beispiel Reaktionsgleichung Bedeutung
Dehydratisierung Ethanol → Ethen CH₃CH₂OH → CH₂=CH₂ + H₂O Herstellung von Ethen, einem wichtigen Grundstoff für die Kunststoffindustrie.
Dehydrierung Ethan → Ethen CH₃CH₃ → CH₂=CH₂ + H₂ Industrielle Herstellung von Alkenen, z. B. für die Polymerisation.
Dehydrohalogenierung 2-Brombutan → But-2-en CH₃CHBrCH₂CH₃ → CH₃CH=CHCH₃ + HBr Synthese von Alkenen aus Halogenalkanen.
Quellen: Chemie-Schule, Wikipedia, Lernhelfer

3. Addition

Definition: Anlagerung eines Moleküls an eine Doppel- oder Dreifachbindung.

Bedeutung: Wichtig für die Synthese von gesättigten Verbindungen, Polymeren und die Herstellung von Alkoholen, Halogenalkanen und vielen weiteren Produkten.

Beispiele & Reaktionsgleichungen

Typ Beispiel Reaktionsgleichung Bedeutung
Elektrophile Addition (Halogenierung) Ethen + Brom → 1,2-Dibromethan CH₂=CH₂ + Br₂ → BrCH₂CH₂Br Herstellung von Halogenalkanen, z. B. für Lösungsmittel oder Zwischenprodukte.
Hydratisierung Ethen + Wasser → Ethanol CH₂=CH₂ + H₂O → CH₃CH₂OH Industrielle Herstellung von Ethanol.
Polymerisation Ethen → Polyethylen n CH₂=CH₂ → (–CH₂–CH₂–)ₙ Herstellung von Kunststoffen (z. B. Polyethylen).
Quellen: Lernhelfer, Abiweb

4. Umlagerung (Isomerie)

Definition: Intramolekulare Wanderung von Atomen oder Atomgruppen, die zu einer neuen Struktur führt.

Bedeutung: Wichtig für die Synthese von Isomeren, die oft unterschiedliche chemische und physikalische Eigenschaften aufweisen.

Beispiele & Reaktionsgleichungen

Typ Beispiel Reaktionsgleichung Bedeutung
Benzidin-Umlagerung Hydrazobenzol → Benzidin C₁₂H₁₂N₂ → C₁₂H₁₂N₂ (Isomer) Synthese von aromatischen Aminen, z. B. für die Farbstoffindustrie.
Isomerisierung von Alkenen But-1-en → But-2-en CH₂=CHCH₂CH₃ → CH₃CH=CHCH₃ Herstellung von Isomeren mit unterschiedlichen Reaktivitäten.
Quellen: Chemie-Schule, Wikipedia

5. Kondensation

Definition: Verbindung zweier Moleküle unter Abspaltung eines kleinen Moleküls (meist Wasser).

Bedeutung: Wichtig für die Synthese von Estern, Polyestern, Peptiden, Proteinen, Polysacchariden und vielen weiteren Biopolymeren.

Beispiele & Reaktionsgleichungen

Typ Beispiel Reaktionsgleichung Bedeutung
Veresterung Ethanol + Essigsäure → Ethylacetat CH₃COOH + CH₃CH₂OH → CH₃COOCH₂CH₃ + H₂O Herstellung von Estern, z. B. für Aromastoffe, Lösungsmittel oder Kunststoffe.
Peptidbindung Glycin + Alanin → Dipeptid NH₂CH₂COOH + NH₂CH(CH₃)COOH → NH₂CH₂CONHCH(CH₃)COOH + H₂O Grundlage der Proteinbiosynthese.
Polykondensation Ethylenglycol + Terephthalsäure → Polyester n HOCH₂CH₂OH + n HOOC–C₆H₄–COOH → (–OCH₂CH₂OOC–C₆H₄–CO–)ₙ + 2n H₂O Herstellung von Kunststoffen wie PET.
Quellen: Wikipedia, Lernhelfer, Chemie-Schule

6. Decarboxylierung

Definition: Abspaltung von Kohlenstoffdioxid (CO₂) aus einer Carbonsäure.

Bedeutung: Wichtig für die Synthese von Alkanen, Alkenen und die Herstellung von Aromastoffen.

Beispiel & Reaktionsgleichung

Typ Beispiel Reaktionsgleichung Bedeutung
Decarboxylierung von Carbonsäuren Essigsäure → Methan CH₃COOH → CH₄ + CO₂ Herstellung von Alkanen, z. B. für die Synthesegaschemie.
Quellen: Lernhelfer

7. Dehalogenierung

Definition: Abspaltung von Halogenatomen aus einer organischen Verbindung.

Bedeutung: Wichtig für die Synthese von Alkenen, Alkinen und die Entgiftung von halogenierten Schadstoffen.

Beispiel & Reaktionsgleichung

Typ Beispiel Reaktionsgleichung Bedeutung
Dehalogenierung von Halogenalkanen 1,2-Dibromethan → Ethen BrCH₂CH₂Br + Zn → CH₂=CH₂ + ZnBr₂ Herstellung von Alkenen, z. B. für die Polymerindustrie.
Quellen: Lernhelfer
